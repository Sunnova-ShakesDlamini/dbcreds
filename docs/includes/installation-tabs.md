=== "pip"

    ```bash
    pip install dbcreds
    ```

=== "uv"

    ```bash
    uv add dbcreds
    ```

=== "pipx (CLI only)"

    ```bash
    pipx install dbcreds
    ```

=== "From source"

    ```bash
    git clone https://github.com/Sunnova-ShakesDlamini/dbcreds.git
    cd dbcreds
    pip install -e .
    ```
